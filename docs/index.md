# Documentation

This documentation provides information about the functions available in the library.


## Usage

```javascript
// CommonJS
const helper = require("melperjs");
const nodeHelper = require("melperjs/node");

// ES Module
import * as helper from "melperjs";
import * as nodeHelper from "melperjs/node";
```

## Available Functions

The library is split into two main modules:
1. Core module (melperjs) - General utility functions
2. Node module (melperjs/node) - Node.js specific functions

See the specific documentation sections for details:
- [General Functions](./general.md)
- [Node.js Functions](./node.md)
